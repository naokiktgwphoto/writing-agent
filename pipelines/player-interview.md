# 選手特集（インタビュー）パイプライン

## インプット

- インタビュー文字起こし（または音声）
- 選手プロフィール（年齢・所属・ポジション・経歴）
- 媒体名・想定文字数・連載か単発か

## 媒体別フォーマット

### Number Web（松澤寛政・吉川大紀など）

**タイトル形式**
```
「引用キーフレーズ」解説テキスト
```
- 例：「なんとかなるよ」レイダースとドラフト外契約、"日本人初"の手前で松澤寛政が見ているもの
- キーフレーズは記事の核心となる選手の言葉から取る。

**章立て**
- `##` で章を区切る。
- 章タイトルは情景・転機・心理状態を示唆する体言止め。
  - 例：「ひとりの春」「指名なき夜」「契約のあと　再び仲間の中で」
- `◇　◇` で時制転換や場面切替を示す。

**文体・構造**
- 冒頭：現在地のシーン → `◇　◇` で時制を遡る、という構成が典型。
- 著者が観察者として地の文に入る。
  - 例：「その落差を『ちょっと』と言えることが、この男の強さだ」
  - 例：「話術ではなく、思考の構造そのものなのだ」
- 締め：余韻・暗示で終わる。状況を断言せず、その先を示唆する。
  - 例：「それはもう手が届く距離かもしれない」

### 4years.（朝日新聞）

対象：学生アスリート・卒業直後の社会人1年目（廣田悠人、安藤謙太、東田隆太郎など）

**タイトル形式**

```
所属・ポジション・選手名　テーマ・キーワード「選手の言葉（引用）」
```

- **選手名・所属を必ず先頭に置く**（テーマから入らない）。
- 引用は末尾に置くか、名前の直後に置く（深掘り特集では名前直後が多い）。
- 例（引用末尾）：「日体大・齊藤進之助　未経験からTOP8の主力RBへ、バレー部出身の異色のキャリア」
- 例（引用末尾）：「京大WR廣田悠人、けがを越え取り戻した感覚「京大史上一番のWRに」」
- 例（引用末尾）：「東田隆太郎、関学から富士通へ「今はまだ10のうち4くらい」新人DBが見据えるもの」
- 例（引用中間）：「京都産業大主将・吉田碧海「自分には自分の人生がある」日本代表の父を持つ男が、アメフトで見つけた道」

**深掘り特集（父子・家族・転機など重層的なテーマ）の場合**

通常の選手プロフィール記事より踏み込んだ構成になる：

- `##` で章立てし、章タイトルにも引用を入れる。
  - 例：「「この4回生と勝ちたかった」3点差が残した傷と動機」
  - 例：「日本代表の父は「ラグビーをやれ」と言わなかった」
- 著者自身が地の文に登場することがある。
  - 例：「取材を終え、グラウンドを出る前に足が止まった」
  - 例：「取材後、車で駅まで送ってもらいながら、明はこう話してくれた」
- 締めは情景描写・余韻型（4years.の標準の「具体的展望」より Number Web的な終わり方）。
- 家族・恩師など選手以外の人物を主要インタビュー対象にする。

**構成**
1. リード（1段落）：選手プロフィール + 今季/今試合の活躍 + テーマ設定 + キーフレーズ（選手の言葉）
2. 具体的なプレーシーンまたは直近の試合エピソードから入る。
3. 経歴・背景（出身・他競技からの転向・家族・きっかけなど）。
4. 成長過程・現状の課題（コーチや周辺人物のコメントを交える）。
5. 今季の位置づけ・チーム状況。
6. 締め：選手の目標・決意の言葉 + 著者の一文で包む。

**引用源**
- 選手本人（メイン）
- コーチ・監督（能力評価や指導方針）：小見出しを「コーチ名「コメントのキーフレーズ」」として独立させることもある
- 家族・先輩・同期（選手の人間性・背景の補強）
- 複数の引用源を組み合わせて立体的に描く。コーチや関係者の長めの引用も厭わない。

**文体**
- 章立て（`##`）は使わず、小見出しで区切る（小見出しは説明的・テーマ要約型）。
- 引用は長めでも可。話し言葉のトーンをそのまま生かす。
- スタッツや試合経過を具体的に記述（ヤード数・クオーター・プレーの詳細）。
- 著者の観察・解釈を要所に差し込む（「〜と苦笑する」「〜が、その言葉の裏に〜があった」）。
- 写真が複数入る前提で、写真に対応する地の文を置く。

### 週刊プレイボーイ（松澤寛政など）

**タイトル形式**
```
【前編】引用フレーズまたはシーン描写。選手名・所属の説明
```
- `【前編】`/`【後編】` を冒頭に置く（前後編の2本立てが基本）。
- 選手名は中盤〜末尾に置く（先頭には置かない）。
- 煽り気味の表現を使うが誇張はしない。
  - 例：「NFLに一番近い日本人になるまで。ハワイ大キッカー・松澤寛政の規格外キャリア」
  - 例：「「外したのに、スタジアム中から拍手が」。ドラフト目前、"Tokyo Toe"松澤寛政が語るキッカーの孤独な仕事」

**構成**
1. リード（2〜3段落）：物語の「起点」となるシーンから入り、現在地のプロフィールを簡潔に置く。
2. 本文：`■` で小見出しを区切る（`##` や `##` は使わない）。
3. 末尾にプロフィールBOX（`〜〜〜` で区切り、氏名・生年月日・経歴を箇条書き的に記載）。

**文体**
- 読者がアメフトを知らない前提で書く。
  - 数字・用語を身近なものに置き換えて説明する。
  - 例：「30ヤードのFGは、野球で言えばイージーなフライのようなもの」
  - 例：「アメフトに詳しくない方のために補足しておくと、……」
- 読者に語りかける地の文を使う。
- Number Webより口語的・エンタメ寄りだが、事実の誇張はしない。
- 著者の解釈・分析を随所に入れる（「〜だからだろう」「〜を物語っている」）。

### BBM Sports（選手特集）

**タイトル形式**
```
テーマを表す一文──選手名、状況の説明
```
- 例：「キッカーの椅子は、ひとつ──松澤寛政、レイダースで挑む53人のロースター枠」
- `──`（ダッシュ）区切りは編集部が付加するスタイル。原稿では省いてよい。
- 選手名は中盤に置く。

**構成**
1. リード（1〜2段落）：現在地 + `【北川直樹】` バイライン。
2. 本文：**章立てなし**。段落内に太字の小見出しを置いて区切る。
   - 例：`指名漏れから契約へ　今感じる「桁違い」の基準`
3. 締め：事実ベースで余韻を持たせる一文。

**文体**
- 引用を多く使う。一問一答に近い密度。
- 著者の観察・解釈を要所に差し込む（「こう言い切れるのは〜からだろう」）。
- 読者がアメフト・競技をある程度知っている前提（週プレほど説明補足をしない）。
- 写真クレジットは `photo by 氏名/チーム名` 形式。

**その他の媒体（Sportiva・BBMなど）**
- 同一題材は切り口・エピソードを各媒体で重複させない。

## 構造の組み立て方（共通）

### 1. 構造軸（テーマ）を一本決める
記事全体を貫く「問い」や「葛藤」を最初に設定する。
- 例：「1.5軍の壁」「経験ゼロからの逆算」「日本を離れる選択」

### 2. 冒頭の入り方
- 具体的なシーン・情景・感覚から入る（抽象的な紹介文から始めない）。
- 例：「飛行機で泣いたのは覚えている」「パスポートを握りしめていた」

### 3. 本文の展開

#### 連載形式（2〜3回）
- 各回に「転機」を一つ置く。
- 時系列で進めながら、核心のエピソードでドラマを作る。
- 中見出しは「出来事＋示唆」の形式。
  - 例：「王子スタジアム、決勝点を蹴った夜」「残り1席だった飛行機」

#### 引用の使い方
- 選手の言葉を短く引用し、その後に著者の解釈を入れる。
- 謙遜・逡巡と確信が同居する瞬間を書く（「謙遜さを挟みながらもそう言い切る」のような観察）。
- 引用は話し言葉のトーンを保つ。整えすぎない。

### 4. 媒体ごとの出し分け
同一取材で複数媒体に書く場合、切り口・エピソードを重複させない。
媒体ごとに使うエピソードを事前に割り振る。

## 禁止事項

- 「——」（em-dash）を使わない。
- 記事の冒頭でプロフィール紹介から入らない（シーンから始める）。
- 締めに「今後の活躍が期待される」などのAI的常套句を使わない。
- 曖昧な意味づけで段落を閉じない（具体的な事実・引用・観察で閉じる）。
