# 試合レポート パイプライン

## インプット

- 実況音声記録（文字起こし）
- 公式ドライブチャート（Time Start/End、得点）
- 公式スタッツ
- HCコメント・選手コメント

## アウトプット構成（この順番で書く）

### 1. タイトル
```
【アメフト】◯◯、△△に◇-◇で勝利　メイントピック
```
- フォーマット厳守。過剰な煽りは避ける。

### 2. リード（1段落）
- 試合概要（スコア・会場・日付）
- MVP的な活躍をした選手
- 両チームのシーズン文脈（それまでの戦績）
- この結果が持つ意義

### 3. 本文（時系列＋小見出し）
おおむね以下の流れ：
1. Q1: 開幕ドライブ・試合の入り
2. Q2: 流れの変化
3. Q3: 突き放す/拮抗
4. Q4: 応戦・クロージング
5. スタッツ解説（本文末尾の独立段落でまとめる）
6. HCコメント・選手コメント

**スタッツ段落の書き方：**
- 総攻撃ヤード・ランヤード・パスヤード・1stダウン数・ターンオーバー数を両チーム比較で記述。
- 個人成績（MVP格の選手のキャリー数・ヤード・TD数など）を含める。
- 3rdダウンコンバージョン率・ボールポゼッションなど文脈に応じて。
- スタッツ段落は「内容では◯◯だったが、ターンオーバーが致命傷」のように試合を再評価する視点を持たせる。

**試合後ハドル・感情シーン：**
- 負けたチームの主将がハドルで語った言葉など、試合後の感情シーンを末尾に追加してよい。
- 書き方例：「試合後のハドルで、◯◯主将は涙を拭いながらチームメイトに語りかけた」

### 4. 結び
- 次戦情報＋HCのフレーミングコメント
- 最終行：試合全体を総括する一文

## 時間表記ルール

- 音声記録は「第◯Q残り◯分◯秒」の残り時間形式だが、記事化は経過時間とする。
- 公式ドライブチャートのTime Start/Endと一致させる。
- 本文中に括弧で時間を補足しない（地の文で経過時間形式を使う）。

## 禁止事項

- `(kick)` などのプレー注記を本文に入れない。
- `(残り◯分)` などの括弧補足を入れない。
- 得点に結びつかないドライブは詳述しない（省略）。ただし、象徴的なプレーやイベントがある場合は記述する。
- 細かいプレーシーケンスの列挙をしない。
- 「アウェー」→「アウェー戦」と書く。

## 媒体別の差分

### Sporting News / Yahoo News

**スコアライン**
リード直後に独立行で置く（BBMと同じ形式）。
```
関西学院大学 ○20-3● 日本大学（MKタクシーフィールド万博）
```

**タイトルの使い分け**

記事の性質によって2種類のタイトル形式を使い分ける。

**① 結果告知型（ストレートニュース）**
```
【アメフト】関学が日大を20-3で下し春季交流戦勝利　WR小段が2TD、守備は4ターンオーバー奪取
【アメフト】立命館が関学を38-14で下し2連覇　QB竹田が4TDに絡む活躍でMVPに
```
- 「◯◯が△△に◇-◇で勝利」の形で試合結果を先に明示。
- サブ情報に「誰が何本のTD」「ターンオーバー数」など数字で補足。
- 速報性を重視する試合レポートに使う（特に大試合・シーズン前半の交流戦など）。

**② 「撒き餌」型（独自取材・コメント主軸）**
```
「シンプルなものの精度が高ければ高いほど強い」明治・中川雄大が語る完勝の理由
なぜ◯◯は強いのか
```
- 選手コメント「」を軸にするか、問いかけ形式にする。
- 長期間読まれる記事・独自取材が主役の記事に使う。
- タイトルの選手名・HC名はフルネーム（日本人でも3〜5文字。名前検索SEO対策）
- 印象的なコメントがあれば「」で使う

**1記事1テーマ、複数記事への展開を検討する**

1回の取材から複数記事に展開できる場合は積極的に分割し、相互リンクを貼る。

| 記事種 | 内容 |
|---|---|
| ストレートニュース | スコア・試合経過。速報性重視 |
| 選手コメント記事 | 独自コメントを軸にした深掘り |
| HCコメント記事 | 独自コメントを軸にした深掘り |
| 分析・深掘り | 戦術・シーズン文脈の考察 |
| フォトギャラリー | 写真集（テキスト最小限） |

大学アメフトは競合メディアが少ない。分割展開＋Yahoo配信でコアファン総取りを狙う。

- 速報性重視。コメントはHC1〜2名を本文末尾にまとめる
- 写真指定なし
- 大学生試合の場合は、名前（学年・出身校）
- 高校生試合の場合は、名前（学年）

### BBM Sports

タイトルはSporting Newsと同形式（`【アメフト】`）。以下がBBM固有：

- 大学生試合の場合は、名前（学年=出身校）
- 高校生試合の場合は、名前（学年）

**スコアライン**
スコアラインはリードの直後に独立行で置く。
```
立命館宇治 ○35-0● 佼成学園（横浜スタジアム、来場者数2938人）
```

**本文の章立て**
試合経過の小見出しは `##` なし（独立した太字行として置く）。
```
序盤から主導権を握った立命館宇治、前半で21-0のリード
```

**コメントセクション**
HC・選手コメントは `##` で個別に章立てし、発言をそのまま掲載する（著者の解釈は挟まない）。
コメントは長くても省略しない。
```
## 立命館宇治・主将QB大鹿蓮のコメント
## 立命館宇治・木下裕介監督のコメント
```
見出しは `## 所属・役職・氏名のコメント` または `## 氏名の話` どちらも可。

**写真**
写真キャプションは `キャプション説明=撮影：北川直樹` 形式。記事冒頭・本文中・末尾に複数挿入。

**バイライン**
リード末尾と記事末尾に `【北川直樹】` を入れる。
