# 試合レポート パイプライン

## インプット

- 実況音声記録（文字起こし）
- 公式ドライブチャート（Time Start/End、得点）
- 公式スタッツ
- HCコメント・選手コメント

## アウトプット構成（この順番で書く）

### 1. タイトル
```
【アメフト】◯◯、△△に◇-◇で勝利　メイントピック
```
- フォーマット厳守。過剰な煽りは避ける。

### 2. リード（1段落）
- 試合概要（スコア・会場・日付）
- MVP的な活躍をした選手
- 両チームのシーズン文脈（それまでの戦績）
- この結果が持つ意義

### 3. 本文（時系列＋小見出し）
おおむね以下の流れ：
1. Q1: 開幕ドライブ・試合の入り
2. Q2: 流れの変化
3. Q3: 突き放す/拮抗
4. Q4: 応戦・クロージング
5. スタッツ解説
6. HCコメント・選手コメント

### 4. 結び
- 次戦情報＋HCのフレーミングコメント
- 最終行：試合全体を総括する一文

## 時間表記ルール

- 音声記録は「第◯Q残り◯分◯秒」の残り時間形式だが、記事化は経過時間とする。
- 公式ドライブチャートのTime Start/Endと一致させる。
- 本文中に括弧で時間を補足しない（地の文で経過時間形式を使う）。

## 禁止事項

- `(kick)` などのプレー注記を本文に入れない。
- `(残り◯分)` などの括弧補足を入れない。
- 得点に結びつかないドライブは詳述しない（省略）。ただし、象徴的なプレーやイベントがある場合は記述する。
- 細かいプレーシーケンスの列挙をしない。
- 「アウェー」→「アウェー戦」と書く。

## 媒体別の差分

### Sporting News / Yahoo News

**タイトルは「撒き餌」型にする**

試合結果の告知ではなく、読者が読みたくなる引き金を作る。独自取材がある記事は特に重要（長期間読まれる）。

```
良い例：「シンプルなものの精度が高ければ高いほど強い」明治・中川雄大が語る完勝の理由
悪い例：明治が立教に27-0で快勝　関東TOP8
```

- 印象的なコメントがあれば「」で使う
- 問いかけ・テーマ提示型も可：`なぜ◯◯は強いのか`
- 結果告知型（`◯◯が△△に勝利`）は速報コンテンツのみ。独自取材記事では避ける
- タイトルの選手名・HC名はフルネーム（日本人でも3〜5文字。名前検索SEO対策）

**1記事1テーマ、複数記事への展開を検討する**

1回の取材から複数記事に展開できる場合は積極的に分割し、相互リンクを貼る。

| 記事種 | 内容 |
|---|---|
| ストレートニュース | スコア・試合経過。速報性重視 |
| 選手コメント記事 | 独自コメントを軸にした深掘り |
| HCコメント記事 | 独自コメントを軸にした深掘り |
| 分析・深掘り | 戦術・シーズン文脈の考察 |
| フォトギャラリー | 写真集（テキスト最小限） |

大学アメフトは競合メディアが少ない。分割展開＋Yahoo配信でコアファン総取りを狙う。

- 速報性重視。コメントはHC1〜2名を本文末尾にまとめる
- 写真指定なし
- 大学生試合の場合は、名前（学年・出身校）
- 高校生試合の場合は、名前（学年）

### BBM Sports

タイトルはSporting Newsと同形式（`【アメフト】`）。以下がBBM固有：

- 大学生試合の場合は、名前（学年=出身校）
- 高校生試合の場合は、名前（学年）

**スコアライン**
スコアラインはリードの直後に独立行で置く。
```
立命館宇治 ○35-0● 佼成学園（横浜スタジアム、来場者数2938人）
```

**本文の章立て**
試合経過の小見出しは `##` なし（独立した太字行として置く）。
```
序盤から主導権を握った立命館宇治、前半で21-0のリード
```

**コメントセクション**
HC・選手コメントは `##` で個別に章立てし、発言をそのまま掲載する（著者の解釈は挟まない）。
コメントは長くても省略しない。
```
## 立命館宇治・主将QB大鹿蓮のコメント
## 立命館宇治・木下裕介監督のコメント
```
見出しは `## 所属・役職・氏名のコメント` または `## 氏名の話` どちらも可。

**写真**
写真キャプションは `キャプション説明=撮影：北川直樹` 形式。記事冒頭・本文中・末尾に複数挿入。

**バイライン**
リード末尾と記事末尾に `【北川直樹】` を入れる。

- [ ] タイトルは指定フォーマットか

- [ ] 外国人選手＝初出フルネーム／以降姓、日本人＝姓のみになっているか
- [ ] Sporting Newsのタイトルは「撒き餌」型か、選手名フルネームか
- [ ] 本文に `(kick)` や括弧補足が残っていないか
- [ ] 詳述したドライブはすべて得点に結びついているか（例外：象徴的なプレー）
- [ ] 「アウェー戦」表記になっているか
- [ ] 曖昧な常套句が混入していないか
- [ ] 結びに次戦情報＋HCコメント、最終行に総括が入っているか
